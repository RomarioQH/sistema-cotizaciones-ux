# Sistema de Cotizaciones

Proyecto estático demo en HTML, CSS, Bootstrap 5 y JavaScript.

## Estructura

- `index.html`: login
- `registro.html`: registro de cliente
- `recuperar-password.html`: recuperación de contraseña
- `pages/admin`: vistas del administrador
- `pages/vendedor`: vistas del vendedor
- `pages/cliente`: vistas del cliente
- `assets/css/style.css`: estilos globales y componentes
- `assets/js/main.js`: utilidades UI
- `assets/js/quote-builder.js`: demo interactiva para cotizaciones

## Recomendación

Abrir con un servidor local para mejor experiencia de navegación.
Por ejemplo con VS Code + Live Server o Aperturar directo del HTML en carpeta.
